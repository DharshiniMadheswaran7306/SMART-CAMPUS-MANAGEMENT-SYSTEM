🔧 How to Fix — Step by Step

1

Enable 2-Step Verification on your Google Account

Go to myaccount.google.com/security → find 2-Step Verification → turn it ON. (This is required before App Passwords will appear.)

2

Create a Gmail App Password

Go to myaccount.google.com/apppasswords
→ App name: type Smart Campus → click Create
→ Google shows a 16-character password like abcd efgh ijkl mnop
→ Copy it (you won't see it again)

3

Update app.py

Open app.py and find these two lines near the top:

EMAIL_SENDER = "your_email@gmail.com"
EMAIL_PASSWORD = "your_app_password"

Replace with your real values:

EMAIL_SENDER = "yourname@gmail.com"
EMAIL_PASSWORD = "abcdefghijklmnop" # 16-char app password, no spaces
4

Restart Flask & test below

Stop the server (Ctrl+C) and run python app.py again. Then use the test form below to verify emails are working.
